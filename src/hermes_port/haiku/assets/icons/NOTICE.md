# Hemera Haiku Icon Notice

The shipped Hemera Haiku app icon and toolbar icons in this directory are derived from MIT-licensed Haiku icon sets.

- Primary shipped source: `darealshinji/haiku-icons` (MIT)
- Secondary visual reference only: `phillbush/haiku-icons`
- Not shipped directly: `retrosmart-icon-theme`

The app icon baseline is the generic Haiku mail icon, used here as a generic mail glyph for Hemera.

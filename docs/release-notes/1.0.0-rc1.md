# Hemera 1.0.0 RC1

Hemera 1.0.0 RC1 packages the Haiku shell with release-facing branding, app resources, and toolbar icons.

## Included in this RC

- Hemera app naming and Haiku resource metadata
- embedded generic Haiku mail app icon
- toolbar icon catalog and staged toolbar HVIF assets
- Haiku package metadata and repo-local HaikuPorts recipe scaffolding

## Remaining runtime-only validation

- compose final dispatch validation on a live Haiku runtime
- Directory Services preview-close/reset validation on a live Haiku runtime

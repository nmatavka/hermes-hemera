# Hemera 1.0

Hemera 1.0 ships the Haiku shell with release-facing branding, app resources, toolbar icons, and the HaikuPorts rollout path.

## Included in 1.0

- Hemera app naming and Haiku resource metadata
- embedded generic Haiku mail app icon
- toolbar icon catalog and staged toolbar HVIF assets
- Haiku package metadata and repo-local HaikuPorts recipe scaffolding
- HaikuPorts rollout tooling for tagged source release, recipe rendering, PR creation, and watch flow

## Known follow-up validation

- compose final dispatch needs broader live Haiku runtime validation
- Directory Services preview-close/reset needs broader live Haiku runtime validation
